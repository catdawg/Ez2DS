\subsection{Detailed Description}
The bounding box needs to be calculated by e2d\-Element\-Calculated\-Bounding\-Box before you can use e2d\-Element\-Center\-At\-B\-Box. In physics engines, the center of mass is usually the origin of the geometries being added to a body. From S\-V\-G it is hard to clearly define this origin. With e2d\-Element\-Center\-At\-B\-Box you can define this origin by using values inside this B\-Box, i.\-e. centering at 0.\-5,0.\-5 will bring the origin to the center of the bounding box.

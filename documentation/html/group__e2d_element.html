Data Structures

struct  e2dElement
 The e2dElement struct is the base of all elements in the scene. Common scene information such as id, attribute values or transformations are saved in the e2dElement struct. You are not meant to instance this struct by itself. By creating other elements such as e2dGroup or e2dImage, this struct will be automatically allocated in them. This "inheritance" is mimic'd by placing the e2dElement struct as the first element in structs which are meant to be "substructs" of it. More...

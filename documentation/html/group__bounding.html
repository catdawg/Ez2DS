Detailed Description

The bounding box needs to be calculated by e2dElementCalculatedBoundingBox before you can use e2dElementCenterAtBBox. In physics engines, the center of mass is usually the origin of the geometries being added to a body. From SVG it is hard to clearly define this origin. With e2dElementCenterAtBBox you can define this origin by using values inside this BBox, i.e. centering at 0.5,0.5 will bring the origin to the center of the bounding box.

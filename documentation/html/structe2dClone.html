Detailed Description

The e2dClone struct is used for holding a pointer to another element in the scene, by having its ID. It is taken from the use tag in SVG, with the exception of the (width,height) attributes being ignored, as they seem to be useless in Inkscape.

e2dClone::pointsToElement is added for convenience. When you load a scene, all clones are dereferenced (using e2dCloneDereference()) and the result is saved here.

It "inherits" from e2dElement by placing it as the first element in the struct, if you typecast e2dClone* into e2dElement*, it will work thus mimicing inheritance in e.g. C++.

Detailed Description

The e2dImage struct is used for holding the paths to images and their size and position. The struct is composed directly from the contents of the SVG image element. It "inherits" from e2dElement by placing it as the first element in the struct, if you typecast e2dStruct* into e2dElement*, it will work thus mimicing inheritance in e.g. C++.

Field Documentation

e2dElement element

e2dImage inherits from e2dElement. Typecasting e2dImage* to e2dElement* will work.

float height

Height of the image, may be -1 if not defined in the SVG

char* imagePath

Path to the image. This is taken directly from the SVG, so if you are getting a local path for example, you need to fix it on the SVG.

e2dPoint position

Position of the image. Note that it is after applying the transformation present in e2dElement::localTransform.

float width

Width of the image, may be -1 if not defined in the SVG

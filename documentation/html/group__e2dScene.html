e2dSceneSearchResult * e2dSceneSearch (e2dScene *scene, char *searchString)
 Currently searches the tree for an element with an ID equal to searchString. Features planned are for searching by type or by attribute and value combination, all with using a path. For example, the search string '**\=tag+image' will return all the images which are children of an element with a tag attribute.
void e2dSceneSearchResultDestroy (e2dSceneSearchResult *ssr)
 Destroys the search result struct. Call on the results of e2dSceneSearch() when you are done with them.

Easy 2D Scene (Ez2DS), is a library for transforming an SVG file into a 2D scene tree with an API. In essence, the library provides the means for applications to quickly have a 2D editor (Any SVG editor).

Detailed Description

Attributes taken from SVG elements are stored here. Useful for when you need to read custom tags placed in the SVG's XML.

Variable Documentation

e2dMatrix effectiveTransform

transform calculated by multiplying all the transformations from the root of the scene until this element

e2dMatrix inverseEffectiveTransform

utility for methods such as e2dElementGetWorldPosition

e2dMatrix localTransform

transform taken directly from the svg. If changed directly, effective transforms will have to be recalculated with e2dSceneCalculateEffectiveTransforms

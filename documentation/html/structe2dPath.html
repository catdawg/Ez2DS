Detailed Description

The e2dPath struct is used for holding the information of path. The struct is composed directly from the contents of the SVG path element. It "inherits" from e2dElement by placing it as the first element in the struct, if you typecast e2dPath* into e2dElement*, it will work thus mimicing inheritance in e.g. C++.
